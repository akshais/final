Features

You can use normal text, headings, buttons, and other things in the Jumbotron. Another feature of Bootstrap is the header bar that is present on every page of this site. The header bar can fit on both a computer screen and a mobile screen because it detects the screen size and then changes to a single button if it is too small. History

Bootstrap was initially called Twitter Blueprint, and was created by Mark Otto and Jacob Thornton. Their goal was to have consistency in web development tools. Before the development of Bootstrap, many different tools were used, which led to inconsistency. It was originated by a small group of developers in order to make a better internal tool.
